Descrição do Projeto

O RachaAí é um aplicativo desenvolvido para facilitar a divisão de despesas entre pessoas que compartilham gastos, como moradores de repúblicas, apartamentos, viagens ou eventos.

O aplicativo permite criar grupos, adicionar despesas, dividir valores automaticamente, acompanhar dívidas e confirmar pagamentos de maneira simples e organizada.

O objetivo principal do projeto é reduzir conflitos e dificuldades relacionados à divisão de contas.

Persona

O aplicativo foi projetado principalmente para estudantes e jovens adultos que dividem apartamento, viagens ou despesas em grupo.

Esses usuários precisam de uma forma rápida, clara e segura de organizar pagamentos e controlar gastos compartilhados.

Heurística de Nielsen em foco

A heurística priorizada foi “Visibilidade do Status do Sistema”.

Durante todo o fluxo, o usuário recebe feedback claro sobre:

valores pagos,
dívidas pendentes,
confirmações de ações,
notificações enviadas,
e status dos pagamentos.

Isso ajuda a transmitir segurança, transparência e controle das informações financeiras.

Telas do Protótipo
1. Dashboard de Grupos

Tela principal do aplicativo, mostrando os grupos criados, saldo geral e notificações sobre pagamentos pendentes.

2. Detalhes do Grupo

Exibe os gastos específicos de um grupo, participantes e histórico de despesas.

3. Adicionar Gasto

Permite cadastrar uma nova despesa, selecionar participantes e escolher o tipo de divisão.

4. Scanner de Nota Fiscal

Tela responsável pela leitura da nota fiscal utilizando a câmera do dispositivo.

5. Extrato de Dívidas

Mostra quem deve para quem, os valores pendentes e opções de pagamento.

6. Confirmação de Pagamento

Tela de feedback exibida após a realização de um pagamento.

Fluxo de Tarefa
Cadastro de uma conta de luz dividida entre 3 pessoas
O usuário acessa o Dashboard de grupos.
Seleciona o grupo “Casa”.
Clica em “Adicionar gasto”.
Informa a descrição “Conta de Luz”.
Adiciona o valor da conta.
Seleciona os participantes.
Escolhe o tipo de divisão igual.
Confirma o gasto.
O sistema calcula automaticamente o valor para cada participante.
O débito aparece no extrato de dívidas.
Justificativa de Design

O layout foi desenvolvido para transmitir clareza e organização, já que lidar com dinheiro pode gerar ansiedade e conflitos.

As informações mais importantes, como valores, dívidas e confirmações, foram destacadas visualmente para facilitar a compreensão rápida.

Além disso:

os fluxos foram reduzidos para poucas etapas;
os botões possuem destaque visual;
as telas apresentam navegação simples e consistente;
e o sistema fornece feedback constante ao usuário.

Essas decisões ajudam a transmitir segurança e transparência mesmo em um protótipo de baixa fidelidade.
